# 💳 Checkout Page UI

A modern and clean checkout page built using HTML and CSS.
This project demonstrates a real-world payment interface with form inputs and order summary layout.

---

## ✨ Features

* Clean and modern UI design
* Responsive layout (mobile-friendly)
* Payment form with structured inputs
* Order summary section
* Gradient call-to-action button
* Soft shadows and spacing for better UX

---

## 🛠 Technologies Used

* HTML5
* CSS3 (Flexbox)
* Google Fonts (Inter)

---

## 📂 Project Structure

```bash
checkout-page/
 ├── index.html
 ├── style.css
```
